Hermes 100: A coding project

Live life one Loop at a time. I created a Javascript coding exorcise using loops. Hermes 100 will display the numbers 0 to 100 and highlight the even numbers in BOLD.

I Built this application using the following tech stack:

  • CSS and BootStrap Layouts
  • Javascript Loops
  • Javascript Functions
  • Javascript DOM manipulation
  • Javascript If|Then|Else
  • Javascript Boolen Logic
  • Javascript Control Flow
